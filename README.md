# Mini Hotel Booking System

A simple DBMS-focused hotel room reservation system using:
- MongoDB
- Express.js
- HTML/CSS/JavaScript

## Features
- Room listing
- Date-based availability check
- Room booking
- View bookings
- Admin booking view

## How to Run
1. Start MongoDB
2. Run backend:
   npm install
   npm start
3. Open frontend/index.html in browser
